# Q-learning Implementation in Tic-Tac-Toe

Q-learning agent learns Tic-Tac-Toe

## Description

I implemented a Q-learning agent that learns to play Tic-Tac-Toe by storing data points that correspond to a certain reward (1, -1, 0) in a Q table from playing hundreds of thousands of games against itself.

## Getting Started

### Usage

```
python runner.py
```

### How It Works

1. Specify how many training matches you want the AI to be playing (to train it),

2. Once done, you can basically play against it.
