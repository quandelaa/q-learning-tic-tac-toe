# Q-learning Implementation in Tic-Tac-Toe

Q-learning agent learns Tic-Tac-Toe

## Description

A Q-learning agent learns to play Tic-Tac-Toe

## Installation

1. Clone the repository and `cd` into it:
```bash
git clone https://github.com/quandelaa/q-learning-tic-tac-toe.git
cd q-learning-tic-tac-toe/
```
2. Run the program:
```bash
python src/runner.py
```
